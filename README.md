 ⚡ 5V Transistor Relay Driver Circuit

📌 Description:
Used to control a relay using a low-power input signal.

🔧 Symbols:
IN   → Input Signal
R    → Resistor
Q    → Transistor (Switch)
RL   → Relay
D    → Diode (Protection)
VCC  → +5V Supply
GND  → Ground

🔁 Working:
IN → R → Q → RL → VCC
             ↓
            GND

D is connected across RL

✔ IN HIGH → Q ON → RL ON → Load ON
✔ IN LOW  → Q OFF → RL OFF → Load OFF

🛡 Diode protects from back voltage
📚 Learning Outcomes:

• Basics of relay operation  
• How a transistor (Q) works as a switch  
• Interfacing low-power signal with high-power device  
• Use of resistor (R) for current limiting  
• Importance of diode (D) for protection (back EMF)  
• Control of external loads using digital signals  
• Understanding practical electronic circuits  

🎯 Summary:
Learned how to safely control a high-power device using a low-power signal.
